### 1. 跨國 e-Invoice API Gateway（2024）

**影響力最高**

- **專案描述：**
因應馬來西亞政府稅務政策，實作 API Gateway 整合高雄廠的馬來西亞分區與馬來西亞廠的發票數據。
- **主要貢獻：**
    - 自主開發 API Gateway 對接馬來西亞政府（LHDN）系統，取代 Oracle 外部解決方案，每年為公司節省數十萬美金的維護費用。
    - 實作複雜的數位簽章功能，確保高雄廠與馬來西亞廠的電子發票傳輸符合當地法規。
    - 協助引導架設 API 站台、資料庫，並順利交接給馬來西亞分區的同仁。

### 2. 高彈性共用系統開發（2023 – 2025）

**AI 應用能力與架構設計**

- **專案描述：**
高彈性共用系統的開發，減少重複開發類似需求的開發工時。額外實作 Gen AI 整合應用於系統功能。
- **主要貢獻：**
    - **共用問卷系統**：參考 SurveyCake，設計高彈性的問卷編輯器與統計視覺化模組（支援多題型、邏輯跳題、即時預覽）。額外整合 ChatGPT API，讓使用者能直接以自然語言描述需求，由 LLM 自動生成問卷草稿；並提供 AI 分析蒐集結果，提升管理決策效率。
    - **共用庫存管理系統**：採用「配置驅動（Config-driven）」架構，透過資料庫設定檔，定義不同業務屬性，實現「一系統多用途」，無需為每個單位重新開發前端/後端邏輯。
    - 降低各部門獨立開發問卷/庫存工具的需求，節省開發工時。問卷可因應不同需求做調查；庫存管理已成功支援酒類、工程實驗品、文書等至少 3 種截然不同的庫存情境，證明架構彈性與可擴展性。

### 3. 採購與福利委員會系統生態圈（2021 – 2022）

快速交付與全端實作能力

- **專案描述：**
完成福委會用的端到端請購、請款系統。將佈告欄與員工福利網轉化為 WebAPP 嵌入企業 App 中。
- **主要貢獻：**
    - 獨立開發涵蓋詢價、比價、訂購、收貨驗收及請款的完整簽核工作流，實現福委會採購流程數位化與規範化。
    - 提升員工在公司外部獲取資訊的便利性。
